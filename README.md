> **Status: Code Coming Soon**

> 

> *The manuscript associated with this repository is currently under peer review. The core scripts will be made publicly available here shortly.*



## Overview



This repository is established to support our manuscript. It will contain the computational system used for the Involuntary Movement (IM) quantification presented in our study.



## Expected Contents



Once the preparation is complete, this repository will include the following core components:



- **Needle Tip Localization:** The core algorithms and scripts designed to robustly identify, extract, and track the spatial coordinates of the needle or instrument tip from raw data.



- **IM Data Quantification:** Analytical scripts utilized to process the localized coordinates and calculate the final quantitative metrics of involuntary movements.
